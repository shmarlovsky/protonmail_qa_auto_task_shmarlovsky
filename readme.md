#### Author: Kirill Shmarlovski. kiril@shmarlovski.by  

#####Requirements:
- python 3 (tested with 3.8.2)
- Google Chrome (tested with Chrome v80)
- python packages: pytest, selenium

##### How to run tests
1. Install dependencies
    - `pip install pytest selenium`
2. Specify account details in `protonmail_auto/account_details.json`
2. Run `pytest`

Note: after execution logs are available at pytest.log
