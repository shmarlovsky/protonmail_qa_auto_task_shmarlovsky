Requirements:
- python 3 (tested with 3.8.2)
- Google Chrome (tested with Chrome v80)
- python packages: pytest, selenium

##### How to run tests
1. Install dependencies
    - `pip install pytest selenium`
    - Download [Chrome driver](https://sites.google.com/a/chromium.org/chromedriver/downloads)
2. Specify account details in config.py > Config > USERNAME, PASSWORD
2. Run `pytest`
